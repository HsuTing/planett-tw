Install

- Install button
npm install planett-button --save
- Install button group
npm install planett-button-group --save



Example

Basic

<Button>Basic</Button>



Change style

<Button style={{background: 'red'}}
        clickedStyle={{background: 'blue'}}
>Change style</Button>



Default clicked

<Button defaultClicked
>Default is Clicked</Button>



Disabled

<Button disabled
>disabled</Button>



Control clicked

You can set clicked by yourself, but you must give a function onClick. Otherwise, you must use defaultClicked.
<Button onClick={ () => { alert('click') } }
        clicked={true}
>Clicked</Button>



Inverted

<Button inverted
>Basic</Button>



Group

<ButtonGroup>
  <Button>Buuton1</Button>
  <Button>Buuton2</Button>
  <Button>Buuton3</Button>
</ButtonGroup>



Default group clicked

default must be an integer. The buttons of group are an array. As a result, default starts from 0.
<ButtonGroup default={1}>
  <Button>Buuton1</Button>
  <Button>Buuton2</Button>
  <Button>Buuton3</Button>
</ButtonGroup>
